# CV Generator

## High Level Design

- Takes input for vaious fields of a resume
- Generates a preview on A4 size canvas
- Option to export as pdf


### Sections

- Personal Details
    - Name
    - Phone
    - Email
    - LinkedIn
- Education
    - List of qualifications from latest to oldest 
    - Each item will have
        - Name of institute
        - Start year/month
        - end year/month
        - name of degree or level or course
        - grade or percentage or cgpa
        - location
- Work Experience
    - each item will have:
        - Name of org
        - start year/month
        - end year/month
        - location
        - designation
        - List of work achievments
- Projects
    - each item
        - title
        - github link
        - date range
        - languages
        - List of achivements
- Skills
    - Languages
    - Frameworks and tools
- Achievemtns
    - lists of achievements
